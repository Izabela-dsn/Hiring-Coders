    • SPA, PWA, Webcomponents

    SPA(Single Page Application), aplicações que concentram seus conteúdos em apenas uma página, quando há modificação na nossa pagina ela não atualiza por completo, somente o que modificamos que atualiza.

    PWA(Progressives Web Apps), aplicações hibridas entre web e mobile, permite o usuário ter a experiencia web com o app instalado no celular. Podendo armazenar as informações em cache e a pessoa pode 'usar' o app offline.

    Webcomponents forma de criar componentes reutilizáveis para web. Fundamentos: Custom elements, shadow dom, html templates, html imports.
